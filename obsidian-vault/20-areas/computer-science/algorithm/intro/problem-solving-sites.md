## 1. 한국 사이트

### 백준 온라인 저지 (BOJ)
- [https://www.acmicpc.net/](https://www.acmicpc.net/)
- 한국 표준 — 27000+ 문제. 한국어 / 영어.
- 난이도: solved.ac (브론즈 → 루비) — 비공식 등급.
- 추천 경로: 단계별 → 알고리즘 분류별 → 기업 출제.
- **백준 단계별 문제**: [https://www.acmicpc.net/step](https://www.acmicpc.net/step)
- **solved.ac**: [https://solved.ac/](https://solved.ac/) — 난이도 / 풀이 통계

### 프로그래머스 (Programmers)
- [https://programmers.co.kr/](https://programmers.co.kr/)
- 카카오 운영. 카카오 / 네이버 / 라인 기출 다수.
- 자바스크립트 풀이 가능 (FE 채용 대비).
- 채용 연계 — 코테 응시 + 채용 매칭.

### SW Expert Academy (삼성)
- [https://swexpertacademy.com/](https://swexpertacademy.com/)
- 삼성 SW 역량 테스트 / 알고리즘 특강.
- 삼성 채용 코테 대비 필수.

### 코드포스 한국
- [https://codeforces.com/](https://codeforces.com/) — 글로벌이지만 한국 유저 많음.

### 알고스팟
- [https://algospot.com/](https://algospot.com/) — "프로그래밍 대회에서 배우는 알고리즘 문제해결전략" 책 동반 사이트.

## 2. 글로벌 사이트

### LeetCode
- [https://leetcode.com/](https://leetcode.com/)
- 글로벌 표준. 3000+ 문제 + 회사 태그 (Google / Meta / Amazon ...).
- **Top 150 Interview Questions**: [https://leetcode.com/studyplan/top-interview-150/](https://leetcode.com/studyplan/top-interview-150/)
- **NeetCode 150**: [https://neetcode.io/](https://neetcode.io/) — 가장 효율적 학습 경로.

### HackerRank
- [https://www.hackerrank.com/](https://www.hackerrank.com/)
- 신입 채용 코테 다수.
- Cracking the Coding Interview 책 연계.

### Codeforces
- [https://codeforces.com/](https://codeforces.com/)
- 알고리즘 대회 표준. 매주 라이브 라운드.
- Rating 시스템 (1200=뉴비 / 1900=expert / 2400=grandmaster).

### AtCoder
- [https://atcoder.jp/](https://atcoder.jp/)
- 일본 대회. 깔끔한 문제 / 한국 / 일본 / 동아시아 유저 많음.
- ABC / ARC / AGC 시리즈.

### Codewars
- [https://www.codewars.com/](https://www.codewars.com/)
- 게임화된 풀이. 입문용으로 좋음.

### HackerEarth
- [https://www.hackerearth.com/](https://www.hackerearth.com/)
- 글로벌 대회 + 채용.

## 3. 학습 큐레이션 사이트

### NeetCode
- [https://neetcode.io/](https://neetcode.io/) / [YouTube](https://www.youtube.com/@NeetCode)
- LeetCode 150 / Roadmap. 시각화 풀이 영상.

### AlgoExpert
- [https://www.algoexpert.io/](https://www.algoexpert.io/) — 유료. 회사 면접 특화.

### Grokking the Coding Interview

- [https://cses.fi/problemset/](https://cses.fi/problemset/) — 알고리즘 학습용 큐레이션 (Finland 책 연계).

### USACO Guide
- [https://usaco.guide/](https://usaco.guide/) — 미국 학생 대회 / 무료 학습 자료.

## 6. 책 / 강의

### 한국어
- 이것이 취업을 위한 코딩 테스트다 (나동빈) — **본 사전의 기준 책**
- 파이썬 알고리즘 인터뷰 (박상길) — LeetCode 풀이 책
- 컴퓨팅 사고력을 키우는 알고리즘 (홍진욱) — 학교 교재

## 7. 추천 학습 경로

### 입문 (1-2 개월)
1. 이코테 책 PART 02 (8 챕터) 정독
2. 백준 단계별 풀기 — 입출력 → 조건문 → 반복문 → 함수 → 배열
3. Programmers Level 1 약 20 문제

### 중급 (2-4 개월)
1. 이코테 PART 03 (8 챕터 기출 45 문제)
2. 백준 실버 ~ 골드 5 약 50 문제
3. Programmers Level 2-3 약 50 문제

### 고급 (4-6 개월)
1. LeetCode 150 (NeetCode 경로)
2. 백준 골드 1 ~ 플래티넘 5
3. 회사별 기출 (카카오 / 라인 / 쿠팡)
4. Codeforces Round 참여 (Div 3-4 시작)
