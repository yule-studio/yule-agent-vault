# Azure Storage (Hub)

**[[../cloud-azure|↑ Azure]]**

| 서비스 | 노트 | 한 줄 |
| --- | --- | --- |
| **Blob Storage** | [[blob-storage]] | Object (S3 동등) |
| **Files** | [[files]] | SMB / NFS (EFS 동등) |

추가:
- **Managed Disks** — VM block disk
- **Queue Storage** — 간단 queue (Service Bus 의 가벼운)
- **Table Storage** — NoSQL (옛, Cosmos DB Table API)

Storage Account = 4 service (Blob/File/Queue/Table) 의 컨테이너.
