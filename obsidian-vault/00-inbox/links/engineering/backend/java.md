## Reference 링크

- [OpenJDK](https://openjdk.org/) — OpenJDK 공식 — 릴리즈 노트 / JEP 인덱스
- [Java Platform SE 21 docs](https://docs.oracle.com/en/java/javase/21/docs/api/) — JDK API 권위 source
- [Effective Java summary (HugoMatilla)](https://github.com/HugoMatilla/Effective-JAVA-Summary) — Effective Java 90 항목 GitHub 요약
- [Modern Java Recipes](https://github.com/kousen/java_8_recipes) — Java 8+ Stream / Optional 예시 코드
- [Java Language Spec](https://docs.oracle.com/javase/specs/) — JLS 공식 — 모호한 동작 출처
- [Java Records 가이드](https://docs.oracle.com/en/java/javase/16/language/records.html) — Java 14+ record 패턴
- [Project Loom (Virtual Threads)](https://openjdk.org/projects/loom/) — Java 21 가상 스레드
- [Java Memory Model — Aleksey Shipilëv](https://shipilev.net/blog/2014/jmm-pragmatics/) — JMM 의 사실상 표준 reference
- [GC Tuning Guide (Oracle)](https://docs.oracle.com/en/java/javase/21/gctuning/) — G1 / ZGC / Shenandoah 튜닝
- [Java Performance — Scott Oaks (책)](https://www.oreilly.com/library/view/java-performance-2nd/9781492056102/) — JVM 성능 표준 책
- [Inside Java (Oracle blog)](https://inside.java/) — JDK 팀 공식 블로그
- [Baeldung Java tutorials](https://www.baeldung.com/category/java/) — Java 실무 패턴 모음
- [JEP 인덱스](https://openjdk.org/jeps/0) — 모든 JEP (Java Enhancement Proposal) 목록
- [Maven 공식 docs](https://maven.apache.org/guides/) — pom / dependency 관리
- [Gradle 공식 docs](https://docs.gradle.org/current/userguide/userguide.html) — 빌드 도구 표준 (Spring Boot 기본)
