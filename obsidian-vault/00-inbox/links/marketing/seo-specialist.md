## Reference 링크

- [Google Search Central docs](https://developers.google.com/search/docs) — SEO 의 권위 source
- [Ahrefs blog](https://ahrefs.com/blog/) — 키워드 / 백링크 케이스
- [Backlinko (Brian Dean)](https://backlinko.com/) — SEO 가이드 (skyscraper 등)
- [Moz blog](https://moz.com/blog) — 도메인 권위 / 링크 빌딩
- [Search Engine Journal](https://www.searchenginejournal.com/) — 업계 뉴스 + 가이드
- [Web.dev — SEO](https://web.dev/learn/seo/) — Google 의 SEO 학습 코스
- [Semrush blog](https://www.semrush.com/blog/) — SEO 도구 + 케이스
- [Schema.org](https://schema.org/) — 구조화 데이터 표준
- [Yoast SEO blog](https://yoast.com/seo-blog/) — 온페이지 SEO 패턴
- [Search Engine Land](https://searchengineland.com/) — SEO 뉴스 + 알고리즘 업데이트
- [Google Core Web Vitals](https://web.dev/articles/vitals) — LCP / CLS / INP 이해

## 운영 규칙

1. 링크 검증 — 죽은 링크는 발견 시 즉시 제거.
2. 새 링크 추가 시 한 줄 설명 (왜 좋은지) 필수.
3. 깊은 학습 자료는 `30-resources/` 에 별도 노트로 추출 + 본 카탈로그에는 링크만.
