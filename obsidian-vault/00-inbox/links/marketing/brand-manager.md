## Reference 링크

- [Aaker on Branding (David Aaker)](https://www.davidaaker.com/) — 브랜드 권위 reference
- [Stripe Press — books](https://press.stripe.com/) — 브랜드 표현의 표준
- [Linear blog](https://linear.app/blog) — 브랜드 + 제품 운영의 사례 (B2B SaaS)
- [Tone of Voice 가이드 (Mailchimp)](https://styleguide.mailchimp.com/voice-and-tone/) — 브랜드 톤 가이드 표준
- [Brand Style Guide examples (Frontify)](https://www.frontify.com/en/blog/best-brand-style-guides) — 브랜드 가이드 케이스
- [Marty Neumeier — The Brand Gap](https://www.martyneumeier.com/the-brand-gap) — 브랜드 전략의 표준 슬라이드
- [Pentagram case studies](https://www.pentagram.com/work) — 브랜드 / 비주얼 아이덴티티 사례
- [Brand New (UnderConsideration)](https://www.underconsideration.com/brandnew/) — 리브랜딩 사례 비평
- [PR Crisis playbook (Edelman)](https://www.edelman.com/insights) — PR 위기 응대 가이드
- [Reputation Institute](https://www.reptrak.com/insights/) — 브랜드 평판 측정
- [Awwwards](https://www.awwwards.com/) — 브랜드 + 웹 디자인 케이스
