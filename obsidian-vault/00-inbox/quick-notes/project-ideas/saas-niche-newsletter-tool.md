## 개요

특정 니치에 한국 최적화 SaaS — 한국어 SEO / 결제 / 본인인증 + 카카오 통합.

## 수익 모델

대부분 $10-50/mo Pro. 한국 시장은 결제 (토스/카카오) + 본인인증 (KCB) + 세금계산서 (홈택스).

## 실제 레퍼런스

채널톡 (한국 SaaS, ARR $100M+) / 자비스 / Stibee / 스티비 / Plani (개인 작업관리). Pieter Levels (12+ SaaS 동시 운영).

## 기술 스택

Next.js + Vercel + Supabase + 토스페이먼츠 + KCB / NICE 본인인증.

## 진입 장벽

한국 시장 = 결제 / 본인인증 / 세금 처리가 진입 장벽. SaaS 자체보다 운영 인프라 학습이 더 시간.

## 한국 시장 변형

한국어 콘텐츠 SaaS / 한국 자영업 SaaS / 한국 학생 SaaS.

## 학습 자료

채널톡 사례 / Pieter Levels playbook (https://levels.io/) / Indie Hackers (https://www.indiehackers.com/)
